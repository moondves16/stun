Green Light

Introduce

Green Light API는 사용자가 지도에서 자신의 주변 음식점 검색 시 이동수단에 따른 소요시간과 식사시간을
고려하여 사용자를 돕는 서비스를 제공하는 API입니다.






적용한 API



HTTP 요청을 사용하여 길 찾기 API를 사용하여 운전, 사이클링, 걷기 및 대중 교통 라우팅에 액세스 할 수 있습니다. 웨이포인트는 특정 위치를 통해 경로를 변경할 수 있는 기능을 제공합니다.

모바일 클라이언트가 Geolocation API로 감지할 수 있는 셀 타워 및 WiFi 액세스 포인트와 같은 정보를 기반으로 위치 및 정확도 반경을 찾습니다.

길찾기 API는 지도 앱 메인 화면에서 주요 지역까지의 소요 시간과 거리, 예상 유류비, 통행 요금 정보를 제공합니다. 그리고 경로상 분기점에서 안내가 필요한 경우 해당 지점에서의 guide code를 제공합니다.


Google 지도에서 사용하는 동일한 데이터베이스에서 데이터를 가져옵니다. 장소는 소유자 검증 목록과 사용자 중재 기여를 통해 자주 업데이트되는 1억 개 이상의 비즈니스와 관심 지점을 특징으로 합니다.
