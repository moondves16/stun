사용 방법

★ 전체 Default 값
-> 모든 파라미터는 입력하지 않아도 default 값이 지정되어 있기 때문에 정상적으로 json을 불러올 수 있습니다. (http://54.180.34.116:5000/api/)

★ 사용자가 원하는 위치의 위도 경도 검색
-> auto 파라미터 사용 시 0을 입력하면 자신이 원하는 위치의 위도 경도를 입력할 수 있습니다. (http://54.180.34.116:5000/api/?auto=0&lat=37.32&lng=127.33)

★ 사용자의 현재 위치 검색
-> auto 파라미터 사용 시 1을 입력하면 Google geolocation API를 사용해 자신의 위치를 사용합니다. (이 기능은 서버에서 사용 시 서버 컴퓨터 기준으로 위치가 잡히기 때문에 API 사용자가 Green Light API를 재가공하여 사용할 때를 고려해 만들어 놓은 기능임) (http://54.180.34.116:5000/api/?auto=1 )

★ 탐색 범위 설정하여 음식점 검색
-> radius 파라미터 사용 시 자신이 탐색을 원하는 반경을 미터 단위로 지정할 수 있습니다. (http://54.180.34.116:5000/api/radius=1000 )

★ 식사시간 파라미터 적용
-> mealstime 파라미터 사용 시 자신의 식사시간을 분 단위로 지정할 수 있습니다. (http://54.180.34.116:5000/api/mealstime=40)

★ 전체 파라미터를 사용자에 맞게 적용
-> 모든 파라미터는 함께 사용가능하며 입력 순서와 무관하게 사용가능하며 auto 파라미터가 1일때는 사용자가 입력한 위도 경도 값이 무시됩니다. (http://54.180.34.116:5000/api/?auto=1&lat=37.32&lng=127.33&radius=1000&mealstime=40 )
